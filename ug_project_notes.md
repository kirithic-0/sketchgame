# Undergraduate Project Journal: Mapgame / Sketchgame

## Project Overview
This repository contains the codebase for my undergraduate (UG) project, centered around an interactive mapping/sketching game (Mapgame). It incorporates a web frontend (React/Vite) and a Python backend with machine learning pipelines for evaluating sketches/images.

## Recent Progress
- **Custom Image Quality Model**: Working on developing a machine learning model to assess image quality. 
- **Annotator Tooling**: Built a custom annotator tool (`tools/annotator`) using Flask to download images and provide a web interface for manual labeling/annotation to build our dataset.

## Current Focus (As of July 2, 2026)
- **Data Augmentation**: Currently focusing on "augmenting and stuff." This involves applying data augmentation techniques to the dataset collected for the image quality model. Expanding the dataset through augmentation will help improve the model's robustness and generalization for the UG project requirements.
